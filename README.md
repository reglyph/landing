# Reglyph Landing Page

A small landing page for @reglyph built with Next.js, React, and TypeScript.

## Quick Start
#### Install dependencies:
```aiignore
npm install
```
#### Production
```aiignore
npm run build
npm run start
```
#### Development
```aiignore
npm run dev
```

## Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Motion
- Google Fonts (JetBrains Mono via `next/font`)

## Requirements
- Node.js >= 18
- npm
